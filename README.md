# Scripta


## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install
```

## Development Server

Start the development server on http://localhost:3000

```bash
yarn run dev
```

## Testing

Run Tests

```bash
yarn run tests
```

Coverage

```bash
yarn run coverage
```



## Production

Build the application for production:

```bash
yarn run build
```

Locally preview production build:

```bash
yarn run preview
```
